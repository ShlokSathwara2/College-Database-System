# College Database System

Full-stack application for managing students, faculty, departments, courses, enrollment, marks, and attendance.

## Stack

- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MySQL

## Project Structure

- `frontend/` - web client
- `backend/` - API server
- `database/` - schema and seed SQL scripts

## Quick Start

1. Set up MySQL and run SQL files from `database/`.
2. Install backend dependencies and create your own `backend/.env`.
3. Install frontend dependencies.
4. Run backend and frontend development servers.

## Security Note

No secrets or private keys are stored in this repository. Create your own local environment file for credentials.
